# Purpose

This document defines the shared vocabulary used throughout the MiEvent handbook.

Consistent terminology reduces ambiguity across product, design and engineering.

---

# Terms

**Organisation** — The highest-level business entity. Every Workspace belongs to exactly one Organisation.

**Workspace** — The core operational unit representing a single event.

**Persona** — An operational user role within the platform.

**Module** — An independent feature area within a Workspace.
