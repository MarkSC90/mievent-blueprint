# Purpose

The Foundations section defines the shared principles, standards and vocabulary that underpin every other volume of the MiEvent handbook.

Where the numbered volumes describe specific areas of the product, Foundations captures the cross-cutting beliefs and conventions that all of them must respect.

Every product, design, architecture, engineering and security decision should be consistent with the documents in this section.

# How to Use This Section

- Read the principles before starting new work in any volume.
- When two approaches conflict, prefer the one that best satisfies these foundations.
- Reference the `glossary.md` to keep terminology consistent across the handbook.
- Treat `technology-stack.md` and `coding-standards.md` as authoritative for implementation.
